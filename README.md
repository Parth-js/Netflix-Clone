# Netflix Clone
https://netflix-pt.vercel.app/  
A Netflix-inspired web application built using React.js, Firebase Authentication, and TMDB API.

## Features

* User Login & Signup with Firebase Authentication
* Browse Popular, Top Rated, Upcoming, and Now Playing Movies
* Watch Movie Trailers from YouTube
* Responsive Netflix-style UI
* Movie Details and Trailer Player
* React Router Navigation
* TMDB API Integration

## Technologies Used

* React.js
* Firebase Authentication
* TMDB API
* React Router DOM
* CSS3
* Vite

## Installation

1. Clone the repository

```bash
git clone https://github.com/Parth-js/Netflix-Clone.git
```

2. Navigate to the project folder

```bash
cd Netflix-Clone
```

3. Install dependencies

```bash
npm install
```

4. Create a `.env` file and add your Firebase and TMDB credentials.

5. Start the development server

```bash
npm run dev
```
